
# QGIS邻区规划工具

基于QGIS的5G/4G邻区规划工具，支持智能图层匹配、空间索引加速、邻区评分（距离+方位角）、结果导出等功能，适用于通信网络规划场景。


## 功能特点
1. **智能图层配置**：自动匹配4G/5G小区图层的字段（基站ID、小区ID、经纬度等）；
2. **高效空间计算**：集成QGIS空间索引，大幅提升邻区查询速度；
3. **多维度评分**：支持“距离+方位角覆盖相关度”的综合邻区评分；
4. **结果管理**：支持CSV导出规划结果、失败清单统计；
5. **兼容多版本**：适配QGIS 3.10及以上版本。


## 使用步骤
1. **图层配置**：在“图层配置”选项卡选择4G/5G点图层，自动/手动映射字段并保存；

![步骤1：图层配置界面](https://github.com/0039fy/cell-neighbor-planner/blob/main/image/1.png?raw=true)

2. **参数配置**：在“参数配置”选项卡设置规划模式（5G→4G/5G→5G/4G→4G）、距离限制、权重等；

https://github.com/0039fy/cell-neighbor-planner/blob/main/image/2.png?raw=true
3. **邻区规划**：在“邻区规划”选项卡输入需要规划邻区的目标小区，支持从剪贴板直接粘贴，点击“开始规划”；


4. **结果查看**：在“结果展示”选项卡查看规划结果，支持导出CSV或复制数据。


## 依赖环境
- QGIS 3.10及以上版本；
- Python 3.7+（QGIS内置Python已满足）。


## 安装方式
1. 打开QGIS，进入“插件管理器”；
2. 搜索“邻区规划工具”，点击“安装”；
3. 安装完成后，在QGIS工具栏/插件菜单中启动工具。
